# ruby-web-scraper

# Features!

  - You can scrap the web content data with this scrapper code


### Installation

Install the dependencies and devDependencies and start the scrap.

```sh
$ cd ruby-web-scraper
$ bundle install
$ ruby scraper.rb
```
